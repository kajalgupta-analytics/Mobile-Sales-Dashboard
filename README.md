📌 Project Overview

This project is a Mobile Sales Data Dashboard created using Power BI.
It provides insights into sales performance, customer behavior, and transaction trends.

🚀 Key Features
Total Sales, Quantity & Transactions analysis
City-wise sales visualization (Map)
Monthly sales trends 📈
Sales by Mobile Model & Brand 📱
Payment method analysis 💳
Customer ratings overview ⭐
